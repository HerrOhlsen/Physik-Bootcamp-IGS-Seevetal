# Physik Bootcamp – SCORM-Pakete

SCORM 1.2 Lernpakete für das Physik Bootcamp (Jg. 11–13).

## Struktur

5 Tracks, 3 Level, 10 Gate Tests:

| Track | Thema | Level 1 | Level 2 | Level 3 |
|-------|-------|---------|---------|---------|
| A | Formeln umstellen | TrackA1* | TrackA2 | TrackA3 |
| B | Rechnen & Einheiten | TrackB1 | TrackB2 | TrackB3 |
| C | Diagramme | TrackC1 | TrackC2 | TrackC3 |
| D | Fachsprache | TrackD1 | TrackD2 | TrackD3 |
| E | Synthese | TrackE1 | TrackE2 | TrackE3 |

*TrackA1 wird separat verwaltet.

**Gate Tests** zwischen den Leveln: GateA1A2, GateA2A3, GateB1B2, ... (je 70% zum Bestehen).

## Moodle-Integration

Pakete können als **Download-Pakettyp** in Moodle eingebunden werden:

1. Admin: `allowtypelocalsync` aktivieren
2. Lernpaket anlegen, Pakettyp "Download" wählen
3. URL eintragen: `https://raw.githubusercontent.com/HerrOhlsen/physik-bootcamp-scorm/main/packages/DATEINAME.zip`
4. Auto-Update-Häufigkeit einstellen

## Dateien

Alle SCORM-Pakete liegen im Ordner `packages/`.
